This is a full-stack application that provides samples for handling data in a relational database with ORM and handling hierarchical data in the NgRx Store with NgRx Data.

Store hierarchical data in normalized form.

The basic concepts of normalizing data are:
  • Each type of data gets its own "table" in the state.
  • Each "data table" should store the individual items in an object, with the IDs of the items as keys and the items themselves as the values.
  • Any references to individual items should be done by storing the item's ID.
  • Arrays of IDs should be used to indicate ordering.

Convert normalized data from store into hierarchical form with selectors.
